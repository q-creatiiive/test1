دورة JavaScript

(JavaScript) جافا سكريبت هي لغة برمجة تُستخدم بشكل أساسي في تطوير تطبيقات الويب التفاعلية.
Tags
لغة برمجة ويب تفاعلية: تُستخدم لتنفيذ منطق معين على جانب العميل (المتصفح).
الأساس في تطوير الويب: تُعد مع HTML و CSS أساسًا لتطوير الويب الديناميكي.

أهداف هذه الدورة

  • التعرف على مفهوم JavaScript
  • تعلم اساسيات JavaScript
  • طرق إضافة النصوص، الفيديوهات وغيرها من محتويات الويب

متطلبات هذه الدورة

  • متصفح سفاري أو قوقل كروم
  • محرر نصوص
  • Visual Studio Code

الخميس المشروع

تُعتبر JavaScript من أكثر لغات البرمجة شيوعًا في العالم، حيث تُستخدم في تطوير واجهات المستخدم، التفاعل مع الصفحات، وإنشاء تطبيقات ويب كاملة. تُمكِّن المطورين من إضافة سلوك ديناميكي للمواقع مثل القوائم المنسدلة، النماذج التفاعلية، والتحديثات الفورية دون إعادة تحميل الصفحة. ومع تطور مكتباتها وأطر عملها مثل React وVue، أصبحت JavaScript أداة أساسية لا غنى عنها لكل مطور ويب حديث.
